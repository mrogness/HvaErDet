# hvaerdet
Norsk-Engelsk for Matilda

Keywords:
$nt - for inline norwegian translation, for ex. $nt jeg liker mat will have the bot respond with "I like food".

$rnt - for reply norwegian transaltion, meaning if you reply to a message that's in norwegian and write $rnt the bot will repsond with the translation of that replied-to message.

Run the main.py file to run the bot
